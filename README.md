# create-nestjs-pg-app

> Scaffold a production-ready [NestJS](https://nestjs.com/) + PostgreSQL project in seconds with zero setup.

![npm](https://img.shields.io/npm/v/create-nestjs-pg-app?color=blue)  
Create a new project based on [`nestjs-production-ready-starter`](https://github.com/GaetanWcz/nestjs-production-ready-starter) using a single command.

---

## 🚀 Quick Start

```bash
npx create-nestjs-pg-app
```


You can directly specfiy the name of your application by adding an extra argument
```bash
npx create-nestjs-pg-app my-awesome-nestjs-application
```

You can skip dependencies installation by adding `--no-install` argument.

```bash
npx create-nestjs-pg-app my-awesome-nestjs-application --no-install
```

---

## 🧰 Features

- 🔧 Based on a production-grade NestJS + PostgreSQL starter
- ⚡ Uses [`degit`](https://github.com/Rich-Harris/degit) to clone the template without Git history
- 🎛️ Choose your package manager (`npm`, `yarn`, or `pnpm`)
- ✏️ Automatically sets the project name in `package.json`

---

## 📝 Options

| Option         | Description                            |
|----------------|----------------------------------------|
| `--no-install` | Skip automatic dependency installation |
